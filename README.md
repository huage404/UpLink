# UpLink

项目初衷是为了解决公司项目有多个服务器环境，每次都需要手动上传文件到服务器，非常麻烦。所以开发了这个工具。

基于 Electron + Vue 3 的桌面端文件上传工具，用于将本地目录通过 FTP / FTPS / SFTP 上传到远程服务器。

## 功能简介

### 首页
- **多协议支持**：普通 FTP、显式 FTPS、隐式 FTPS、SFTP（SSH）
- **服务器管理**：添加、编辑、删除服务器配置，按主机分组展示
- **目录上传**：选择本机路径与服务器路径，递归上传整个目录
- **过滤规则**：支持正则或简单通配（如 `*.log`）排除指定文件
- **导入导出**：支持从 JSON 文件或粘贴内容导入/导出配置，便于备份与迁移
- **上传进度**：实时进度与当前文件名，支持取消上传、查看上传日志

### 任务管理
- **创建任务**：输入任务名称，选择首页已添加的多个服务器
- **批量上传**：一键将任务上传到所选全部服务器
- **上传日志**：每个任务支持查看详细上传日志

## 应用结构

左侧菜单栏导航：
- **首页**：服务器列表与单次上传
- **任务管理**：创建任务并批量上传到多个服务器

## 技术栈

- **Electron** + **Vite**：桌面应用与构建
- **Vue 3** + **Vue Router**：前端框架
- **Element Plus**：UI 组件
- **basic-ftp**：FTP/FTPS 连接
- **ssh2-sftp-client**：SFTP 连接

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发运行
pnpm start

# 打包
pnpm package

# 制作安装包
pnpm make
```

## 系统要求

- Node.js 18+
- 支持 Windows、macOS、Linux
